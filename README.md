# Netmeds-Clone

A Collaborative project completed in five day along with four Web Developers.

- Tech Stack :
***HTML | CSS | Javascript***

Features
=======
>- Home Page & Navbar
>- Login / Register functionality
>- Products List Page
>- Various types of filtering & sorting
>- Products Details Page
>- Cart Page
>- Shiping Page
